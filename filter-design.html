The Design Problem

From Specification to Coefficients

Filter design starts with a specification: "pass frequencies below 1 kHz with less than 0.5 dB ripple; reject frequencies above 1.2 kHz by at least 60 dB." The design problem is: find the minimum-order filter (fewest coefficients) that meets this spec. Different methods make different trade-offs in how they distribute the approximation error.

IIR Design: Classical Analogue Prototypes

The Analogue-to-Digital Pipeline

Classical IIR design leverages a century of analogue filter theory. The process:

  1. Design an analogue prototype lowpass filter Ha(s)
  2. Apply frequency transformations (lowpass → highpass, bandpass, etc.)
  3. Convert to digital via the bilinear transform: s = 2fs(z-1)/(z+1)

The bilinear transform maps the entire analogue frequency axis to the digital frequency range [0, π], with a slight frequency warping that must be pre-compensated.

Butterworth: Maximum Flatness

Stephen Butterworth, 1930

The Butterworth filter has the flattest possible magnitude response in the passband — no ripple at all. It's defined by having all derivatives of |H(jΩ)|² equal to zero at Ω=0 (maximally flat). The trade-off: it requires the highest order of any classical filter type for a given transition width.

  • Passband: Maximally flat (monotonically decreasing)
  • Stopband: Monotonically decreasing
  • Roll-off: 20N dB/decade = 6N dB/octave (N = order)
  • Poles: Equally spaced on a circle in the s-plane (Butterworth circle)
  • -3 dB point: Exactly at the cutoff frequency
|H(jΩ)|² = 1 / (1 + (Ω/Ωc)2N) Butterworth magnitude-squared response

A Butterworth filter is like a perfectly maintained highway that's flat for miles, then smoothly descends a hill. No bumps, no ripples — just a gentle, predictable decline. If you need steeper descent, add more lanes (higher order), but the road always stays smooth.

Chebyshev: Controlled Ripple for Steeper Roll-off

Type I: Equiripple Passband

The Chebyshev Type I filter allows controlled ripple in the passband (typically 0.5–3 dB) in exchange for a steeper transition than Butterworth at the same order. The ripple is "equiripple" — it oscillates between the same upper and lower bounds throughout the passband. Named after Pafnuty Chebyshev (1821–1894), whose polynomial approximation theory underlies the design.

Type II (Inverse Chebyshev): Equiripple Stopband

The Chebyshev Type II is the mirror image: flat passband (like Butterworth) but equiripple in the stopband. Less commonly used because most applications tolerate passband ripple better than imperfect stopband rejection.

Elliptic (Cauer): Optimal Transition Width

Wilhelm Cauer, 1930s–1940s

The elliptic filter achieves the sharpest possible transition band for any given order by allowing equiripple in BOTH passband and stopband. For a given set of specifications (passband ripple, stopband attenuation, transition width), the elliptic filter requires the lowest order — the fewest coefficients. The cost: the most non-linear phase response and the most complex design mathematics (involving elliptic functions).

The Bilinear Transform

Analogue to Digital

The bilinear transform maps the s-plane to the z-plane via:

s = (2/T) · (z - 1)/(z + 1)   or equivalently   z = (1 + sT/2)/(1 - sT/2) Bilinear transform

Frequency Warping

The bilinear transform compresses the infinite analogue frequency axis into [0, π]. This creates a non-linear mapping: ωdigital = 2 arctan(ΩanalogueT/2). The remedy: pre-warp the analogue cutoff frequency before design so that the warped version lands at the correct digital frequency. For a desired digital cutoff ωc: Ωc = (2/T) tan(ωc/2).

FIR Design Methods

Window Method

Simple but Suboptimal

Start with the ideal frequency response (a rectangle in frequency), compute its IDFT to get the ideal impulse response (a sinc), then truncate/window it to M+1 samples. The window shape controls the trade-off between main-lobe width (transition band) and side-lobe level (stopband rejection). Simple to implement but doesn't achieve the minimum order for a given spec.

Parks-McClellan: Optimal Equiripple FIR

Parks, McClellan & Rabiner, 1972–1973

The Parks-McClellan algorithm (based on the Remez exchange algorithm) designs the shortest FIR filter that meets a given specification. It distributes the approximation error optimally: equiripple in both passband and stopband, with the maximum error minimised (minimax criterion). This is provably optimal — no shorter FIR filter can meet the same specs.

  • Input: Passband edge, stopband edge, passband ripple, stopband attenuation, sample rate
  • Output: Minimum-length symmetric FIR coefficients
  • Optimality: Achieves the theoretical minimum order for given specs
  • Limitation: Only designs linear-phase (symmetric) FIR filters

Designing a filter with the window method is like cutting a tailored suit with scissors — quick but imprecise, with frayed edges. Parks-McClellan is like having a laser cutter that optimises every stitch to use the minimum fabric while meeting exact measurements. It takes more computation upfront but produces the most efficient result possible.

Least-Squares Design

Minimising Total Error

While Parks-McClellan minimises the maximum error (peaks), least-squares design minimises the total squared error (average). This produces filters with lower average error but potentially higher peaks in some bands. Useful when you want the best overall approximation rather than guaranteed worst-case bounds. Solved via linear algebra (Toeplitz systems).

Practical Design in Real Applications

Audio: Linkwitz-Riley Crossovers

Professional audio crossovers use Linkwitz-Riley (LR) alignment: two cascaded Butterworth filters of order N. The result is -6 dB at crossover (not -3 dB), ensuring the summed power of high and low channels is flat. LR4 (4th order, 24 dB/oct) is the industry standard for active loudspeaker management.

Communications: Root-Raised-Cosine

Digital communications use matched root-raised-cosine (RRC) filters at transmitter and receiver. Convolved together, they produce a raised-cosine response with zero inter-symbol interference (ISI). The roll-off factor α (0 to 1) trades bandwidth for time-domain decay rate. α = 0.35 is common in LTE; α = 0.22 in DVB-S2.

Medical: Linear-Phase Requirements

ECG processing demands linear-phase filters to preserve waveform morphology — the ST segment's shape is diagnostic, and phase distortion can mimic pathology. FIR filters with 0.05 Hz highpass (very long — thousands of taps at typical sample rates) ensure no clinically significant distortion.

Classical filter design optimises for mathematical criteria (ripple, transition width, phase linearity). But human perception isn't linear — we're more sensitive to some distortions than others. "Perceptual filter design" could use psychoacoustic models (or learned perceptual metrics) as the optimisation criterion. A filter that's mathematically suboptimal but perceptually transparent might use fewer coefficients than one designed to minimax specs. Similarly, for machine learning inference pipelines, filters could be optimised for downstream classification accuracy rather than spectral flatness.

Some work exists in perceptual audio coding (MP3, AAC use psychoacoustic masking) but general-purpose perceptual filter design methods remain unexplored as a formal discipline.
