# Digital Signal Processing — Knowledge Base

**[View the live site →](https://xizor1970.github.io/dsp-knowledge-base/)**

An interactive, interconnected educational website exploring Digital Signal Processing from first principles through to production systems. Dark-themed, visually immersive, with hands-on Canvas animations.

## Chapters

| # | Topic | What you'll learn |
|---|-------|-------------------|
| 01 | **Signals & Systems** | Continuous vs. discrete, LTI properties, impulse response |
| 02 | **Sampling & Aliasing** | Nyquist-Shannon theorem, anti-aliasing, quantisation |
| 03 | **Fourier Transform** | DFT, FFT algorithm, frequency-domain thinking |
| 04 | **Convolution & Correlation** | Flip-shift-multiply-sum, fast convolution, matched filters |
| 05 | **Z-Transform** | Poles/zeros, transfer functions, stability |
| 06 | **Digital Filters** | FIR vs. IIR, biquads, linear phase, trade-offs |
| 07 | **Filter Design Methods** | Butterworth, Chebyshev, elliptic, Parks-McClellan |
| 08 | **Windowing & Spectral Analysis** | Leakage, window functions, STFT, spectrograms |
| 09 | **Multirate Processing** | Decimation, interpolation, polyphase, oversampling |
| 10 | **Applications** | Audio, 5G, medical, radar, hardware landscape |
| — | **Interactive Animations** | 5 hands-on visualisations with real-time parameter control |

## Interactive Animations

- **Sampling & Aliasing** — see aliasing happen as you push past Nyquist
- **Fourier Synthesis** — build square/sawtooth/triangle waves from harmonics
- **Convolution** — watch the flip-shift-multiply-sum unfold step by step
- **Pole-Zero Explorer** — drag poles/zeros, see frequency response reshape
- **Window Functions** — compare leakage across rectangular, Hann, Hamming, Blackman
